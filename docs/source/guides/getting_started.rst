Getting Started
===============

.. toctree::

Installation from Source
------------------------

1. Clone the repository:

   .. code-block:: bash

      git clone https://github.com/jmount1992/procdocs.git
      cd procdocs

2. Create and activate a virtual environment:

   .. code-block:: bash

      python -m venv venv
      source venv/bin/activate  # On Windows: venv\Scripts\activate

3. Install the package locally:

   .. code-block:: bash

      pip install -e . # or `pip install -e .['dev']` for developers
